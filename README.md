# Themeparks Api

根据 [Themeparks](https://github.com/cubehouse/themeparks) 封装的乐园等候时间查询 Api

## 主要特性：
- TypeScript
- 按地区读取，查询更便捷
- 增加乐园缩写代码，更利于存储
- 统一的游乐设施标识
